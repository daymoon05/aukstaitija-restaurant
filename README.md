# Aukštaitija Restaurant Operating System

A full-stack restaurant management platform designed to streamline restaurant operations through QR-based ordering, real-time order tracking, and staff management workflows.

## Overview

This project was built to improve restaurant efficiency by providing a seamless ordering experience for customers while giving waiters, kitchen staff, and managers real-time visibility into restaurant operations.

## Features

### Customer Experience
- QR code menu access
- Digital food ordering
- Live order status tracking
- Mobile-friendly interface

### Waiter Dashboard
- Manage active tables
- Track customer orders
- Monitor order progress
- Improve service response times

### Kitchen Management
- Real-time incoming orders
- Order preparation workflow
- Status updates for completed orders
- Reduced communication delays

### Restaurant Operations
- Table management
- Order tracking
- Analytics dashboard
- Operational monitoring

## Tech Stack

### Frontend
- Next.js 14

- Tailwind CSS
